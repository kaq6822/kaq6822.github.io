01.소개

서버가 왜 그렇게 동작하는지를 끝까지 파고드는 것을 좋아하는 백엔드 엔지니어입니다. REST API 서버 설계부터 데이터 모델링, 배포까지 서비스의 뼈대를 만드는 일을 해왔습니다.

요즘은 AI 에이전트가 실제 업무를 안전하게 수행할 수 있게 하는 에이전트 인프라에 몰두하고 있습니다. 권한 통제와 감사 로그를 갖춘 MCP 서버를 직접 설계·구현하며, LLM을 온디바이스에서 구동하는 실험도 병행합니다.

백엔드가 본업이지만 Flutter로 웹과 앱을 만들어 아이디어를 끝까지 제품으로 완성해 보는 사이드 프로젝트도 꾸준히 하고 있습니다.

최근에 주로 다루는 기술들입니다.

  • Go
  • Java
  • Python (FastAPI)
  • TypeScript
  • Flutter / Dart
  • MCP · LLM 인프라
